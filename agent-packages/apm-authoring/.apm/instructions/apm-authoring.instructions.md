## Skill trigger: `apm-authoring`

When editing agent skills, instructions, prompts, `AGENTS.md` or `CLAUDE.md` — or when laying out a new `agent-packages/<name>/` directory — apply the `apm-authoring` skill.

In a repo that uses APM, do not edit `AGENTS.md` / `CLAUDE.md` directly: they are generated by `apm compile`. Edit the matching primitive under `.apm/instructions/` (or `.apm/skills/<name>/SKILL.md`) and re-run `apm compile`.
