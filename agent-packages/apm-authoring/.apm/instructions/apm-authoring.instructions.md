## Skill trigger: `apm-authoring`

When editing any APM primitive (instructions, skills, prompts, agents, hooks, `apm.yml`) or `AGENTS.md` / `CLAUDE.md` — or when laying out a new `agent-packages/<name>/` directory — apply the `apm-authoring` skill.

In a repo that uses APM, do not edit `AGENTS.md` / `CLAUDE.md` directly: they are generated by `apm compile`. Edit the matching primitive under `.apm/` and re-run `apm compile`.
