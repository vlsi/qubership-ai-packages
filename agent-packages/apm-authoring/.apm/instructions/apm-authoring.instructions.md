## Skill trigger: `apm-authoring`

When creating, editing, or reviewing an APM primitive (`apm.yml`, `SKILL.md`, `*.instructions.md`, `*.prompt.md`) — or when laying out a new `agent-packages/<name>/` directory — apply the `apm-authoring` skill.
