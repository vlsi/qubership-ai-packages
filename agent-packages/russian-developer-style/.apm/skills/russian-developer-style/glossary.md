# Глоссарий ИТ-терминов

Стартовый набор русских соответствий для повседневной IT-лексики. Не
продуктовая терминология — её добавляйте в свой проектный глоссарий и
проверяйте отдельным линтером.

Колонка **Перевод?**:

- **да** — переводим всегда;
- **нет** — оставляем по-английски;
- **контекст** — зависит от документа, проекта или сложившейся практики.

Не вносите в этот файл продуктовые термины (имена сервисов, фичей, ролей).
Заведите для них отдельный проектный глоссарий и проверяйте линтером:
часть слов окажется уникальной для проекта и устареет вместе с ним.
